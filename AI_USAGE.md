# AI Usage Disclosure

## What I used AI for

I gave Claude Code well documented prompt with all the requirements for the service. Good and solid prompt can reduce the usage (by planning and implementing once) and also reduce time of development.
After validating and testing the first iteration, I gave Claude feedback with improvemnet to implement. This iterative process of prompt -> feedback -> implementation led to the final result.

Claude did its job good but needs some guidance to implement edge cases.

## What I wrote myself without AI

## My assessment of AI usefulness on this task

I beliecve the use of Claude Code reduce the development time significantly. It leaves time for me to plan and test better.
